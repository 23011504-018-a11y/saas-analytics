# Goals Loop

This loop turns SaaS metrics into a repeatable operating rhythm.

## 1. Set a goal
Choose one primary objective for the next review period, such as:
- Reduce logo churn from 33.3% to below 20%
- Improve NRR from 82.1% to above 100%
- Raise activation and usage so fewer accounts become at risk

## 2. Define the metric
Track a few measurable signals:
- Primary metric: MRR, ARR, logo churn, revenue churn, NRR
- Leading indicators: usage score, support tickets, days since last login

## 3. Diagnose the gap
Review the latest data and ask:
- Which segments are underperforming?
- Which customers are most at risk?
- What changed compared with the prior month?

## 4. Act on the highest-leverage lever
Examples:
- Re-engage low-usage accounts
- Improve onboarding and activation milestones
- Add expansion offers to healthy accounts

## 5. Review and reset
Repeat weekly or monthly, and update the target based on the latest trend.
